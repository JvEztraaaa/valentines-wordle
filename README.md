## 📱 How to Play

1. Click the splash screen heart to begin
2. Guess the 5-letter word in 6 attempts
3. Use the on-screen keyboard or your physical keyboard
4. Color feedback:
   - 🟩 **Green:** Correct letter, correct position
   - 🟨 **Yellow:** Correct letter, wrong position
   - ⬜ **Gray:** Letter not in word


## 💕 Customization

To customize this game for your own Valentine:

- Edit the `WORD` variable in `script.js` to change the main word to guess
- Edit the `LOVE_WORD` variable to change the love question answer
- Modify the splash screen text and styling in `index.html`
- Replace `flower.png` with your own bouquet image
- Change the final message in the bouquet page section
- Update the background color scheme in `styles.css`
